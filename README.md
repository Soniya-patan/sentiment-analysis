# Sentiment Analysis Project

This project predicts whether a movie review is positive or negative using Machine Learning and NLP.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression

## Features
- Predicts positive and negative reviews
- Uses IMDB movie review dataset
- Accuracy around 89%

## How to Run

Install libraries:

pip install pandas scikit-learn

Run project:

python app.py
